Projects

Deep Learning Models for Preliminary Autism Spectrum Disorder Screening

Final Year Project (PFE) | May 2025

Project Overview

This project focused on developing deep learning models for the preliminary screening of Autism Spectrum Disorder (ASD) using image classification techniques. The research aimed to create accessible tools that could assist in early detection, particularly in regions with limited specialized expertise.

Technical Approach

I implemented and compared four CNN architectures: DenseNet121, MobileNetV2, VGG19, and InceptionV3. The approach involved applying transfer learning and fine-tuning techniques to optimize performance, developing ensemble learning methods using soft voting to improve accuracy, and building on Python/TensorFlow with development on Kaggle.

Key Results

  • Achieved 89% accuracy with ensemble learning approach
  • Reached 87.7% accuracy with MobileNetV2 after fine-tuning
  • Demonstrated significant improvement (6.33%) through fine-tuning techniques
  • Optimized models for potential mobile deployment

Model Performance Comparison

Impact

The project aims to facilitate early access to ASD screening, especially in regions where specialized expertise is limited. The AI models serve as complementary tools to traditional clinical methods, potentially improving early intervention rates.

Future Directions

The research establishes the technical foundation for a future mobile application. Next steps include transforming these models into a concrete solution to improve healthcare access.
